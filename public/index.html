使用与评分说明

一句话原则：站内只计算真实返回的推荐词信号；趋势、社媒和电商页面用来交叉复核，不制造演示数据。

怎么用

  1. 先输入大词，例如 Christmas、summer bbq、pet home。
  2. 看热词发现，找平台共同出现的具体词。
  3. 点任意关键词，站内会展开多平台需求信号。
  4. 需要确认季节性时，再打开 Google Trends、Pinterest、TikTok 等入口复核。

怎么交叉验证

  • 同一个词同时出现在搜索、电商、视觉或内容平台，可信度更高。
  • Google Trends 主要看季节峰值和长期走势。
  • Pinterest / Images 主要看视觉风格和场景表达。
  • Amazon / Shopping / eBay 主要确认购物需求和规格词。

怎么发现新思路

  • 从大类词开始，让系统自动拆成产品、颜色、材质、场景和人群。
  • 优先看“产品词 + 使用场景 + 材质/颜色”的组合。
  • 遇到节日词时，看未来 6 个月是否有相容节日或生活场景。
  • 把高分词拿到平台复核，确认是否真有内容和商品供给。

评分和颜色

绿色：实时抓取 / 平台已出现 黄色：仅人工验证 红色：当前被平台限制 灰蓝：建议接入 / 待验证

可信等级

  • A 级：趋势历史 + 购物 + 社媒/视觉 + 3 个以上平台。
  • B 级：至少 2 个平台，且有购物或内容平台支持。
  • C 级：单平台线索，进入观察池，不直接当选品建议。
  • D/E 级：人工入口、受限、无结果或不可判断。

真实边界

  • 推荐词 ≠ 搜索量 ≠ 销量。
  • 不是官方搜索量，也不是销量。
  • 登录页只能采集你当前可见内容。
  • 没有原始链接、采集时间或证据链，不参与评分。
  • 受限平台会进入冷却，不会继续硬刷。
  • 最终选品结论要看趋势、供给、价格带和评论痛点。
